### Type Number:
* Integer:
  * Example: 100;
* Negative Integer:
  * Example: -100;

* Float:
  * Example: 10.52;
* Negative Float:
  * Example: -10.52;

* Hexadecimal:
  * Example: 0xfff;
* Exponential:
  * Example: 256e-5;

* Octal:
  * Example: 030;
* Binary:
  * Example: 0b0010001;

### Type String:
* String Literal:
  * Example: 'Hello World';
* String literal assigned to a variable:
  * Example: const str1 = 'Hello World';

* String Object:
  * Example: let str = new String();
  * str = 'Hello World";

### Type Boolean:
* Boolean Variables:
  * Example: let YES = true;
  * Example: let NO = false;
